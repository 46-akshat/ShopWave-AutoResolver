Phase 2: The Reasoning Engine (CLI)
Goal: Wire up Gemini and LangGraph to create an agent that explains its decisions and handles the mock failures.
State Definition: Define the LangGraph State dictionary (e.g., tracking the ticket details, intermediate tool outputs, and current confidence level).
Tool Binding: Bind your mock tools to the gemini-1.5-pro model.
System Prompting: Craft a strict system instruction demanding the agent log its internal monologue and self-evaluate its confidence before taking action.
The ReAct Loop: Build the core LangGraph nodes:
Agent Node: Evaluates the ticket and decides which tool to call (or if it should escalate).
Tool Node: Executes the tool. Crucially, wrap this in a try/except block to catch the errors from Phase 1 and return them to the agent as "observations" so it can recover.
